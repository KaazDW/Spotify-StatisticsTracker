## About The Project

Spotify Statistics Tracker, web app made in TypeScript and VueJS made for consult your spotify listening statistics.

<br/>

## Getting started
### Download github copy

```sh
git clone https://github.com/KaazDW/Spotify-Stats-Tracks.git
cd Spotify-Stats-Tracks
```

### Install dependencies

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
